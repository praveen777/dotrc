## Vim Editor Shortcuts	
	
| Keyboard shortcut | Function |
| --- | --- |
| `CTRL+j` | To search templates avilable in the file |
| `,g` | To do multifile search in the **present folder** |
| `,/` | To search for a word in a file and jump to that line |
